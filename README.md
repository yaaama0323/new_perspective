## New perspective

Laravel 自作


## 概要
ゲームレビュー投稿サイトを作成しました

一般ユーザー、管理者ユーザーそれぞれでログインできます

## 使い方
一般ユーザー

レビューの検索、レビュー投稿、レビュー情報更新、削除機能,プロフィール設定ができます

管理者ユーザー

他ユーザーのレビューを削除できます


### 

## 環境
MAMP/MySQL/PHP/Laravel

## データベース

データベース名：new_perspective

migrationファイルに格納してありますのでphp artisan migrateにて作成して下さい。
